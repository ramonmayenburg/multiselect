# Documentation

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### run jest unittests
```
npm run test:unit
```

### Deploy to github pages : https://ramonmayenburg.github.io/multiselect/
```
npm run deploy
```
